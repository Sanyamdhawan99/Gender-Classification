Gender Classification

Gender classification problem is an active research area which has attracted a great deal of attention recently. Gender classification problem finds plethora of applications in real world - monitoring, surveillance, commercial profiling and human-computer interaction to name a few.
Here we provide you with a machine learning model which classifies an input image into male or female.
This model takes into account the distinct features like distance between eyes and lips, eyes and nose tips, eyebrows, width of jaw and nose.
